# Output Format

Agar hasil antar role rapi dan konsisten, gunakan format ini.

## Researcher

```markdown
## RESEARCHER
- Ringkasan masalah:
- Analisis inti:
- Opsi solusi:
- Asumsi / catatan:
```

## Critic

```markdown
## CRITIC
- GOOD:
- WEAK:
- NEEDS_IMPROVEMENT:
- RISK:
- Saran revisi:
```

## Orchestrator

```markdown
## ORCHESTRATOR
- Objective:
- Strategi:
- Sintesis:
- Final answer:
```

## Kenapa format ini penting?

Karena tanpa format baku:
- hasil tiap role gampang berantakan
- susah dibandingkan
- susah digabung
- user jadi nggak kebayang value dari tiap agent
