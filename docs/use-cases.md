# Use Cases

Berikut beberapa contoh penggunaan starter multi-agent ini.

## 1. Research Team

Cocok untuk:
- riset topik
- drafting artikel
- menyusun rangkuman
- validasi hasil akhir

Flow:
- `main` menerima task
- `orchestrator` memecah objective
- `researcher` menyusun bahan
- `critic` mengulas hasil
- `main` kirim jawaban final

## 2. Project Planning Assistant

Cocok untuk:
- memecah requirement
- menyusun roadmap
- menganalisis opsi solusi
- review trade-off

Role yang paling berguna:
- `orchestrator`
- `researcher`
- `critic`

## 3. Answer Drafting + Review

Cocok untuk:
- drafting jawaban panjang
- ngecek kelemahan isi
- meningkatkan kualitas jawaban final

## 4. Learning Playground

Cocok buat user yang baru belajar:
- konsep multi-agent
- cara kerja role-based agent
- setup OpenClaw multi-agent dasar

## Catatan

Repo ini lebih fokus ke fondasi dan kemudahan setup, bukan sistem otomatis penuh. Tapi dari fondasi ini, kamu bisa naik ke sistem yang lebih advanced.
